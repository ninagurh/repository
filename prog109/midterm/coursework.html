What did you know before this class?

I was introduced to HTML in 2002 in elementary school. I believe XHTML was used during that time. After that, I continued to use basic knowledge of HTML to edit my own MySpace and Xanga websites after that. My background was very limited.

What skills did you gain from this class?

I learned to format HTML with CSS and JavaScript, something I have never done before. Using code to design different types of documents including resumes, games, and forms. All of which apply to real life.

What knowledge did you gain after this class?

The textbook really distinguished the differences between some of the HTML versions, which help me understand which codes were "old" and "new". I also never created a website from scratch. ...why hello there!
